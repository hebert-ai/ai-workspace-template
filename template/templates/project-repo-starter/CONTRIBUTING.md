# Contributing

## Purpose

This repository accepts contributions for the `<PROJECT_NAME>` project.

## Before Opening a Change

- review the project-local `AGENTS.md`
- review `PROJECT.md` and `ROADMAP.md`
- understand the current milestone before making broad changes

## Pull Requests

- keep changes focused
- explain the purpose of the change
- note any follow-up work or known limitations
- update project docs when behavior changes

## Validation

Run the relevant project checks before submitting changes when practical.

Document what you ran in the pull request description.
