# Claude Workspace Adapter

Read these files in order before doing substantive work from this workspace root:

1. `AGENTS.md`
2. `WORKSPACE.md`
3. `custom/AGENTS.md` if present

This directory is for cross-project planning, organization, and project bootstrap. Do not default to project-specific implementation from the root.

If the user asks to start a new project here, create `projects/<slug>/` and initialize it from `templates/project-starter/`, then keep project-specific work inside that new folder.
