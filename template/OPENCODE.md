# OpenCode Workspace Adapter

This workspace is the shared control plane for multiple projects.

Read first:

1. `AGENTS.md`
2. `WORKSPACE.md`
3. `custom/AGENTS.md` if present

Use the root for high-level planning, reusable resources, and project initialization. When asked to start a project, create `projects/<slug>/` and seed it from `templates/project-starter/`.
